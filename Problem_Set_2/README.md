# Objective
Build a machine learning model to predict whether a customer will subscribe to a term deposit using their banking and demographic data. Logistic Regression is used for this binary classification task.
# Dataset
Used the Bank Marketing Dataset with 17 features including customer details and account information.
Target variable:
Yes → Subscribed
No → Not Subscribed
# Methodology
Preprocessing: Data cleaning, encoding, train-test split
Model: Logistic Regression
Training: Model trained and evaluated using accuracy
# Results
Model predicts customer subscription successfully
Accuracy:89.88%
